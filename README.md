# Assignment: Web Page Using HTML/CSS

## Live Web Page
You can view the live assignment web page [here]((https://symb-assigment.vercel.app/)).


## Description
This project is a simple web page built using **HTML** and **CSS**.  
The design follows the reference UI provided [here](https://symb-assessment.gitlab.io/ui-assignments/assignments/assignment_2.html).

### Features
- **Clear and visible input box** for user interaction.
- **Structured layout** on a white background.
- **Highlighted button** for the primary call-to-action (CTA).
- **Other CTAs** remain unaffected by the styling.

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Clone or download the repository.
2. Open the `index.html` file in your web browser.
3. The page should render the input box, button, and structured layout as designed.
